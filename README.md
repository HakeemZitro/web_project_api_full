# Tripleten web_project_api_full (Project 19)

Aplicación Web para la red social de Around the U.S., donde al registrarte podrás subir tus fotos favoritas, editar tu perfil y dar like a las publicaciones de todos los usuarios. La aplicación esta desplegada en una Maquina Virtual de Google Cloud, utilizando un DNS propio, con certificados SSL y utiliza React para el FrontEnd y Node.js con Express para el BackEnd.

Visita la aplicación en: https://around.hzitro.dev


## Características del proyecto
### FrontEnd
- Vite
- HTML / CSS / JS
- React / JSX

### BackEnd
- NodeJS
- ExpressJS
- REST API
- MongoDB

### Herramientas
- GIT
- GITHUB

### Funcionalidades
- Componentes Funcionales
- React Hooks (useState / useEffect / useContext / useRef)
- React Router (Routes / Route / Navigate / Link / useNavigate / useLocation)
- Funciones Asíncronas para solicitudes HTTP a REST API
- Validacion de formulario Register y Login en FrontEnd con Regex
- Autenticación FrontEnd y BackEnd
- Validacion de datos en BackEnd con Joi y Celebrate
- Manejo centralizado de errores en BackEnd
- Registro de logs en archivos .log
- Configuracion y despliegue de la aplicacion en una Maquina Virtual de Google Cloud
- Certificados SSL
